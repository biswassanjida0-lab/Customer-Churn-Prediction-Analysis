# Customer Churn Prediction & Analysis

## Overview
This project predicts telecom customer churn using Machine Learning techniques.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-Learn
- XGBoost
- SHAP
- Streamlit

## Features
- Customer churn prediction
- Feature importance analysis
- SHAP explainability
- Model evaluation
- Interactive dashboard

## Results
The XGBoost model achieved strong performance in churn prediction.

- app.py
- Dataset files
- Model metrics
