# 📊 Customer Churn Prediction & Analysis

## Dataset

The dataset contains telecom customer information such as:

- Gender
- Senior Citizen
- Partner
- Dependents
- Tenure
- Internet Service
- Contract Type
- Payment Method
- Monthly Charges
- Total Charges

## Models Used

1. Logistic Regression
2. XGBoost
